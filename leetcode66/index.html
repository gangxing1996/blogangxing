Leetcode 66. 加一

Posted on 2022-12-04 In Coding , Leetcode

题目描述

给定一个由 整数 组成的 非空 数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

 

示例 1：

输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。

示例 2：

输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。

示例 3：

输入：digits = [0]
输出：[1]

 

提示：

  • 1 <= digits.length <= 100
  • 0 <= digits[i] <= 9

解答

就普通模拟法，如果最后多了一位，就产生一个新的vector
实际上还可以剪枝，加到不进位就直接return

Complexity

  • Time: $O(n)$
  • Space: $O(1)$

题外话

原来vector可以直接赋值哦，而且经过测试不会内存溢出，但是里面要是套太多东西，估计还是要溢出。
